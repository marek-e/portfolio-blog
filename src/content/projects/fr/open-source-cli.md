CLI open-source créé pour résoudre mon propre problème de configuration répétitive de nouveaux projets. Au lieu de copier des configs entre projets, DevKit les génère selon vos préférences.

Fonctionnalités incluant des prompts interactifs avec Ink pour une UI terminal moderne, une architecture de plugins pour l'extensibilité, et la détection automatique des configurations existantes pour éviter les conflits.
