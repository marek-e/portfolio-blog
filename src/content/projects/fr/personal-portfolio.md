J'ai construit ce portfolio pour présenter mon travail et partager mes réflexions sur le développement web. Le site utilise l'architecture "islands" d'Astro pour des performances optimales, chargeant le JavaScript uniquement là où l'interactivité est nécessaire.

Les décisions techniques clés incluent l'utilisation de l'espace colorimétrique OKLCH pour un thème perceptuellement uniforme, l'implémentation d'un mode sombre sans flash avec persistance localStorage, et la création d'un système de design glassmorphique fonctionnant dans les deux thèmes.
